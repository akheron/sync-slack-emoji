## sync-slack-emoji

[![Greenkeeper badge](https://badges.greenkeeper.io/pine/sync-slack-emoji.svg)](https://greenkeeper.io/)

## Requirements

- Node `$(cat .node-version)`
- Yarn

## Getting started

```
# Edit config file
$ cp config/default.js config/local.js
$ vim config/local.js

# Run script
$ yarn
$ yarn start
```
